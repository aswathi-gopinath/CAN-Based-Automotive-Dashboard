# CAN-Based Automotive Dashboard

## Overview
This project implements an automotive dashboard system using Controller Area Network (CAN) communication. The dashboard receives vehicle data through CAN messages and displays real-time parameters such as speed, RPM, gear status, and warning indicators.

## Features
- CAN message transmission and reception
- Automotive message ID decoding
- Real-time dashboard display using CLCD
- Interrupt-driven CAN communication
- Timer-based task scheduling
- Fault and warning indication

## Technologies Used
- Embedded C
- CAN Protocol
- PIC Microcontroller
- Interrupt Handling
- Real-Time Embedded Systems

## System Working
1. CAN node transmits vehicle data frames
2. CAN controller receives message interrupt
3. Message handler decodes CAN ID
4. Dashboard parameters updated on CLCD

## Hardware Components
- PIC Microcontroller
- CAN Transceiver
- CLCD Display
- Automotive CAN Simulation Setup
